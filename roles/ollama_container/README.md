# ollama_container

## Назначение

Роль для развертывания Ollama в Docker-контейнере на Ubuntu-хосте.

## Что делает

- проверяет, что ОС — Ubuntu;
- проверяет наличие Docker и NVIDIA Container Toolkit;
- создает каталоги под Ollama;
- создает Docker network для LLM-сервисов;
- шаблонит docker compose файл;
- запускает контейнер Ollama;
- проверяет, что контейнер запущен;
- проверяет доступность API Ollama через `/api/tags`.

## Что не делает

- не подтягивает модели;
- не запускает Open WebUI;
- не разворачивает Garak;
- не настраивает reverse proxy.

Это будет вынесено в отдельные роли.
