# ollama_pull_models

## Назначение

Роль для синхронизации списка моделей Ollama внутри уже запущенного контейнера.

## Что делает

- проверяет, что контейнер Ollama существует и запущен;
- проверяет доступность API Ollama;
- получает текущий список моделей через `ollama list`;
- сравнивает текущий список со списком из переменных;
- подтягивает только недостающие модели;
- при необходимости умеет форсировать повторный pull;
- выводит финальный список моделей.

## Что не делает

- не разворачивает контейнер Ollama;
- не устанавливает Docker;
- не настраивает GPU.

Это должно быть выполнено предыдущими ролями.
