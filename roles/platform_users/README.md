# platform_users

## Назначение

Роль для создания и управления административными пользователями LLM-платформы на Ubuntu-хосте.

## Что делает

- проверяет, что список пользователей задан;
- проверяет наличие групп `sudo` и `docker`;
- при необходимости создает общий group для команды;
- создает пользователей;
- задает им password hash;
- добавляет их в `sudo`, `docker` и общий group;
- при необходимости создает общий рабочий каталог.

## Что не делает

- не настраивает SSH keys;
- не настраивает Ansible Vault автоматически;
- не удаляет пользователей.

Это можно добавить позже отдельным расширением роли.
